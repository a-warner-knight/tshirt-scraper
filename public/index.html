Image Overlay Editor

Position and scale design images over base images

Instructions

1. Select a base image and design image from the dropdowns above.
2. The design image will appear centered and scaled to 1/4 of the base image size.
3. Drag the design image to position it where you want.
4. Use the corner handles to resize the design image.
5. The blue box shows the target area where the design should be positioned.
6. Click "Store Position" to save the current scale and position.
